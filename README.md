# Sales Data Assistant

A simple sales data analysis application built with **Python, Pandas, and Streamlit**.  
This project allows users to upload a CSV file and ask business-related questions about sales performance.

## 🚀 Live Demo

🔗 **Try the app here:**  
https://beyza-sales-assistant.streamlit.app

## Features

- Upload CSV sales data
- Ask questions in natural language-like format
- Get quick analytical answers
- View the uploaded dataset directly in the app
  
## Example Questions

- Ortalama satış nedir?
- En yüksek satış nedir?
- En düşük satış nedir?
- Toplam satış nedir?
- En çok satış yapan şehir hangisi?
- Aylara göre satış nasıl?
- En iyi satış elemanı kim?

## How to Run
1. Clone the repository:
git clone https://github.com/beyzauzun-ai/sales-data-assistant.git
2. Go to the project folder:
cd sales-data-assistant
3. Install dependencies:
pip install -r requirements.txt
4. Run the app:
streamlit run app.py

## Project Purpose

This project was built as a hands-on beginner/intermediate data analysis and AI-style application.
It demonstrates how Python and Pandas can be combined with Streamlit to create an interactive business-focused data assistant.
